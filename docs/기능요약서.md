# 도장관 (DOJANGWAN) 기능 요약서

> 태권도 도장 전용 SaaS 플랫폼
> 도메인: `tkd.genomic.cc` | 배포: Cloudflare Workers
> 최종 업데이트: 2026-03-12

---

## 서비스 개요

도장관은 태권도 도장 원장님을 위한 All-in-One SaaS 플랫폼입니다.
원생 관리, 출석 체크, AI 품새 채점, 원격 코칭, 대회 중계까지 하나의 서비스로 제공합니다.

---

## 3-Layer 기능 구조

```
Layer 1: 도장 SaaS (B2B)          ✅ 완료
Layer 2: AI 품새 + 원격 심사/연수   🔨 부분 완료
Layer 3: 대회 라이브 중계           🔨 부분 완료
```

---

## Layer 1: 도장 SaaS ✅

### 1-1. 회원 가입 / 로그인

| 기능 | 설명 |
|------|------|
| 도장 가입 | 도장명, 관장명, 이메일, 비밀번호 입력 → 도장 + 계정 동시 생성 |
| 로그인 | 이메일 + 비밀번호 → JWT 쿠키 발급 (7일 유지) |
| 인증 보호 | 대시보드 전체에 서버 컴포넌트 인증 가드 적용 |
| 통합 로그인 | `.genomic.cc` 도메인 통합 쿠키 (SSO 기반) |

---

### 1-2. 대시보드 홈 (통계)

- 총 원생 수
- 오늘 출석 인원
- 이번 달 신입생 수
- 활성 공지사항 수

---

### 1-3. 원생 관리

| 기능 | 상세 |
|------|------|
| 원생 등록 | 이름, 생년월일, 전화번호, 보호자 전화, 띠, 입학일, 메모 |
| 원생 검색 | 이름 실시간 검색 |
| 띠별 필터 | 흰띠 / 노란띠 / 초록띠 / 파란띠 / 빨간띠 / 검은띠 |
| 상태 필터 | 활동 / 휴무 |
| 단증 정보 | 단·품·급 등급, 국기원 ID, 단증 번호(TKP-연도-도장코드-일련번호), 취득일 |
| NFT 단증 | Polygon SBT 연동 예정 (token_id, tx_hash 필드 보유) |
| 얼굴 이미지 | R2에 업로드 → AI 앨범 자동 분류에 활용 |

---

### 1-4. 출석 관리

| 기능 | 상세 |
|------|------|
| 날짜 이동 | ← → 버튼으로 날짜 이동, 오늘 기준 기본 표시 |
| 원생별 출결 | 출석 / 결석 / 조퇴 선택 |
| 메모 | 원생별 당일 메모 입력 |
| Optimistic UI | 클릭 즉시 반영 (서버 동기 병행) |

---

### 1-5. 공지사항

| 기능 | 상세 |
|------|------|
| 공지 등록/수정/삭제 | 제목 + 내용 |
| 고정 공지 | 상단 고정 (is_pinned) |
| 정렬 | 고정 공지 먼저, 이후 최신순 |

---

### 1-6. 앨범 & AI 사진 분류

| 기능 | 상세 |
|------|------|
| 앨범 생성 | 제목, 행사 날짜, 유형(훈련/대회) |
| 사진 업로드 | R2 저장, 다중 업로드 지원 |
| AI 얼굴 인식 | 원생 사진과 자동 매칭, 신뢰도 점수 포함 |
| 공유 링크 | share_token 기반 공개 앨범 링크 생성 |
| 카카오 알림톡 | 분류된 원생 보호자에게 사진 알림 발송 |

---

### 1-7. 라이브 코칭 (WebRTC)

| 기능 | 상세 |
|------|------|
| 세션 생성 | 제목, 개인/그룹 선택, 최대 참가자 수 설정 |
| 초대 링크 | invite_token 기반 링크 생성 |
| WebRTC 영상 | 1:1 또는 1:N 실시간 영상 통화 |
| 자동 녹화 | 세션 종료 후 R2에 자동 저장 |

---

### 1-8. 도장 설정

| 기능 | 상세 |
|------|------|
| 도장 정보 | 도장명, 주소, 지역(17개 시/도) 수정 |
| 비밀번호 변경 | 현재 비밀번호 확인 후 변경 |
| 요금제 관리 | 플랜 조회 및 변경 (예정) |

---

## Layer 2: AI 품새 + 원격 심사/연수 🔨

### 2-1. AI 품새 채점

| 기능 | 상세 |
|------|------|
| 실시간 자세 분석 | MediaPipe Pose로 카메라 영상 분석 |
| 동작별 점수 | 각 품새 동작에 대한 자세 정확도 점수 |
| 비디오 녹화 | 채점 영상 자동 저장 |
| 무료 체험 | `/poomsae/free` — 로그인 없이 체험 가능 |
| 기기 성능 감지 | 저사양 기기 자동 감지 → 경량 모드 전환 |

---

### 2-2. 원격 승단 심사

| 기능 | 상세 |
|------|------|
| 심사 세션 생성 | 심사 날짜, 대상 등급, 심사위원 지정 |
| 심사위원 화면 | WebRTC 영상 수신 + 실시간 점수 입력 패널 |
| 응시자 화면 | 카메라 전송 + AI 채점 피드백 표시 |
| 초대 링크 | 응시자에게 링크 발송 → 로그인 없이 참가 |
| 결과 저장 | AI 점수 + 수동 점수 혼합 최종 저장 |
| 단증 검증 | `/verify/[certNumber]` — 공개 검증 페이지 |

---

### 2-3. 사범 원격 연수

| 기능 | 상세 |
|------|------|
| 연수 세션 생성 | 강사 1명이 세션 생성 |
| 강사 화면 | 1:N 라이브 방송 + 연수생 피드백 수신 |
| 연수생 화면 | 강사 영상 수신 + 실시간 질의응답 |
| 점수 기록 | 연수생별 참여도/점수 기록 |

---

## Layer 3: 대회 라이브 중계 🔨

### 3-1. 대회 방송

| 기능 | 상세 |
|------|------|
| 대회 생성 | 대회명, 날짜, 참가자 등록 |
| 방송 화면 | 주최자용 WebRTC SFU (여러 카메라 통합) |
| AI 자세 오버레이 | 선수 영상에 Pose 분석 실시간 표시 |
| 실시간 점수판 | 심판 입력 → 시청자에게 실시간 반영 |

### 3-2. 대회 시청

| 기능 | 상세 |
|------|------|
| 다중 영상 시청 | 여러 선수 영상 동시 시청 |
| 공개 접근 | 로그인 없이 링크만으로 시청 가능 |

### 3-3. 대회 사진 (AI 얼굴 인식)

| 기능 | 상세 |
|------|------|
| 사진 업로드 | 대회 전체 사진 일괄 업로드 |
| 선수별 자동 분류 | AI 얼굴인식으로 선수별 사진 자동 분류 |
| 자동 발송 | LINE/카카오로 선수 본인 사진 자동 전송 |

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | Next.js 16 (App Router) |
| 스타일 | Tailwind CSS v4 (모바일 우선) |
| 인증 | JWT (jose) + Cloudflare 쿠키 |
| DB | Cloudflare D1 (SQLite 14개 테이블) |
| 스토리지 | Cloudflare R2 (사진, 영상) |
| AI | MediaPipe Pose + Cloudflare AI |
| 실시간 | WebRTC (Cloudflare Realtime SFU) |
| 배포 | Cloudflare Workers + GitHub Actions |
| 알림 | 카카오 알림톡 (NHN Cloud) |
